福州到拉萨旅游攻略整合包

推荐打开：福州到拉萨旅游攻略_网页整合版.html

包含：
1. 网页整合版 HTML：攻略正文 + 实时天气/7日预报 + 每日行程决策 + 预约准备清单 + 预算计算器 + 可缩放高德地图 + 真实导航折线路径 + 景点图片。
2. 交互路线地图 HTML：专门查看 R1-R9 路线和高德导航入口。
3. PDF 打印版。
4. Word 可编辑版。
5. lhasa_real_maps：网页和文档使用的地图图片、景点图片等资源。

注意：
- HTML 文件需要和 lhasa_real_maps 文件夹保持在同一目录下，图片才能正常显示。
- 实时天气来自 Open-Meteo，无需账号；联网时自动刷新，离线时仍可查看静态攻略。
- 地图底图来自在线高德瓦片，打开网页时需要联网。
- 整合版地图会读取 lhasa_real_maps/routes 下的路线 JSON，在线部署或本地 HTTP 访问时显示真实导航折线；直接双击 HTML 时浏览器可能拦截本地 JSON 读取。
- 路线距离/时间来自高德 MCP 查询，实际出行以高德 App 当天导航为准。
